# UI Designer Skill

Use this skill to apply premium design patterns inspired by Stripe, Vercel, Linear, and Apple.

## Style Guidelines

### 1. High Readability & Typographic Polish
* Match header weights using `font-semibold` or `font-bold` coupled with `tracking-tight`.
* Ensure proper line heights: `leading-relaxed` for paragraphs, `leading-none` or `leading-tight` for titles.
* Group context labels together with small gap boundaries (e.g., `space-y-1` or `gap-1`).

### 2. Cards & Containers
* Apply modern shadows: standard shadow outlines (`shadow-xs` or `shadow-sm`) over container borders.
* Standardize borders: use uniform border widths (`border border-zinc-200` or `border-zinc-800` in dark mode).
* Keep rounded corners consistent with `rounded-xl` (`10px`).

### 3. Subject-Specific Theming
* Maintain theme lookups:
  * **Sleek Slate (default / UI Programming):** Clean container card structures, soft colors, subtle shadows.
  * **Neo-Brutalist (Python Programming):** Sharp boundaries, thick flat borders (`border-4 border-black`), drop shadows (`shadow-[8px_8px_0px_0px_rgba(0,0,0,1)]`).
